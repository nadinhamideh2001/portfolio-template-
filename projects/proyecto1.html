PROJECT

AIORA — Emotional Ring

AIORA es un anillo emocional que mezcla joyería y bienestar. Visualmente, un sistema limpio y sensorial: tipografía cuidada, degradados suaves y detalles metalizados.

Concepto

El proyecto busca una estética íntima y tecnológica, donde la interfaz y el objeto se sientan coherentes. Composición editorial, aire, y un tono emocional sin verse “cargado”.

Sistema pensado para escalar: key visuals, composiciones de producto, y reglas simples para mantener consistencia.
